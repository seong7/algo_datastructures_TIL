## Big O notation

<ol>
    <li>
        O(1)&nbsp; : Constant (상수_ no loop)
    </li>
    <li>
        O(log N)
    </li>
    <li>
        O(n)&nbsp : Linear (n item 을 for, while loop 하는 경우
    </li>
    <li>
        O(n log(n))
    </li>
    <li>
        O(n^2)
    </li>
    <li>
        O(2^n)
    </li>
    <li>
        O(n!)&nbsp : n factorial (절대 사용되면 안됨 / 모든 요소 각각에 loop 를 추가하는 경우)
    </li>
</ol>
